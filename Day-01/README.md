
# Day 01 – Introduction to Linux & Basic Commands

## 🐧 What is Linux?
Linux is an open-source operating system used in servers, cloud platforms,
supercomputers, and embedded systems. It is known for its stability,
security, and command-line power.

Most servers on the internet run on Linux, which makes learning Linux
commands very important for developers, system administrators, and
DevOps engineers.

---

## 💻 Why Use Linux Commands?
Linux commands are used to interact with the system through the terminal.
They help users perform tasks like viewing files, navigating directories,
and managing system resources efficiently.

---

## 📂 Display Files and Directories

### 🔹 ls
Used to display files and directories present in the current location.

```bash
ls


📝 Note:
This command shows only the names of files and folders in the current directory.

🔹 ls -lt

Used to display files and directories in long listing format,
sorted by the latest modified time.

ls -lt


📝 Note:
This command is useful when you want to see recently modified files
along with details like permissions and size.
